# Smart Lottery System 🎲

A simple and smart lottery system built with Python.

## Description
This project runs a random lottery between a list of names or numbers. 
The cool part is that it automatically saves the results to a `.txt` file with a timestamp.

## How to Run
```bash
python lottery.py
```
Then follow the steps:
1. Enter the names or numbers. Type `ok` when you are done
2. Enter the number of winners
3. Type `y` if you want to save the result to a file

## Technologies Used
- Python 3
- `random` library
- `datetime` library
- `os` library
